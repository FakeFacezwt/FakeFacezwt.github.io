常用快捷键整合

2021-09-22 我的第一篇文章

Typora操作快捷键、

  1. 有序表ctrl + shit+【

  2. 无序表ctrl + shit + ]

  3. ctrl + 1 一级标题

    ctrl + 2 二级标题 类推

  4. 加粗 ctrl + B

  5. 插入公式 ss

  6. 分隔符 ***

  7. ctrl + T 做表格

GIt快捷键

  1. git glabol –user name ;git glabol –user.email 设置邮箱和用户名

  2. git init 初始化

  3. 文件操作
    新建文件：vim 文件名
    查看文件：cat 文件名
    查看文件末行：tail -n 行数 文件名
    编辑文件中 esc退出插入，i 插入，yy复制 p粘贴 ；（:wp）保存

  4. git add 文件名 ：添加到缓存区

  5. git status 查看文件夹状态

  6. git rm –cached 文件名 移除缓存区文件

  7. git commit -m 提交日志 文件名

  8. git reset –hard 精简版本号 去到指定版本号

  9. 分支
    创建分支：git branch 分支名
    切换分支：git checkout 分支名

    查看分支：git branch -v

    合并分支：git merge 分支名

    冲突产生的时候直接更改源文件就好

  10. github 仓库

  11. 给仓库地址创建别名：git remote add 别名 仓库地址

  12. 推送代码到远程仓库：git push 仓库地址或者别名 要推送的分支

  13. 从远程仓库复制代码： git clone 仓库地址 会自动生成origin别名
    想推送更改代码到原仓库 需要原仓库户主把你拉进团队，授权以后才可以推送

想要团队协作
1.需要公司把你的账户授权，然后就可以看到对方的仓库就可以进行协作了
2.外部团队协作需要fort分叉 然后pull request就可以了

npm操作

  1. npm config set registry https://registry.npm.taobao.org
    npm install

IDEA常用快捷键

  1. Ctrl + R 在当前文件进行文本替换
  2. Ctrl + Y 删除光标所在行 或 删除选中的行 （必备）
  3. Ctrl + D 复制光标所在行 或 复制选择内容，并把复制内容插入光标位置下面 （必备）
  4. Ctrl + / 注释光标所在行代码，会根据当前不同文件类型使用不同的注释符号 （必备）
  5. Ctrl + Alt + L 格式化代码，可以对当前文件和整个包目录使用 （必备）
  6. Ctrl + Alt + O 优化导入的类，可以对当前文件和整个包目录使用 （必备）
  7. Ctrl + Shift + Enter 自动结束代码，行末自动添加分号 （必备）
